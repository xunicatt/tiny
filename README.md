# tiny

A tiny clock made for fun!

### Hardware

- stm8s103f3 (mcu)
- tm1637 (4 digit - 7 segment display)
- ds3231 (rtc)
- ldr (for auto brightness)
- buzzer (for alarms)
- 3 status led
    + green (brightness controllable)
    + red
    + green (brightness controllable))
- 3 push button
- IR proximity sensor (for waking up from sleep)
- Type-C port for power
